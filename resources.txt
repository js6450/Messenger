Things to install:
Node.js: go to their website and install the right version for your computer
After you have installed node.js, install cordova and ionic by typing:
npm install -g cordova ionic

on your terminal (for mac) or cmd (for windows)

for mac, if that doesn't work, try
sudo npm install -g cordova ionic

emulation tool:
We need to be able to emulate how the app will look like on our computer.
Since we are developing for android, we need to install this emulation tool:
http://genymotion.com/

Download and install their free version.
